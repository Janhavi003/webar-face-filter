# Real-Time Camera Filters (Web)

A clean, mobile-friendly camera filters demo built with modern web APIs.
The project focuses on **real-time video rendering, subtle post-processing effects, and polished UX**, rather than gimmicky AR overlays.

Designed as a **portfolio-quality frontend project**, it demonstrates camera access, canvas-based rendering, and thoughtful UI architecture.

---

## Features

- Real-time camera feed (front-facing)
- Subtle, photographic filters:
  - Neutral
  - Warm / Cool
  - Contrast
  - Soft Blur
  - Black & White
  - Matte
  - Vivid
- Clean camera permission flow
- Custom filter menu UI (no native dropdowns)
- Mobile-friendly and responsive
- High-performance rendering via `<canvas>`

---

## Tech Stack

- JavaScript (ES Modules)
- Web Camera API (`getUserMedia`)
- HTML5 Canvas
- Modern CSS (glass UI, responsive layout)

No backend required.
