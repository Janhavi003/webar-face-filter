# WebAR Face Filter

A mobile-friendly, real-time face filter built with JavaScript, MediaPipe Face Mesh, and Three.js.

## Features
- Real-time face tracking
- Face-anchored AR effects
- Clean camera permission UX
- Mobile-first performance

## Tech Stack
- JavaScript (ES Modules)
- MediaPipe Face Mesh
- Three.js
- Web Camera API
